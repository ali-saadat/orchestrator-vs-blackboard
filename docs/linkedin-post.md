# LinkedIn post — Supervisor vs. Blackboard article

Goal: the article shows as an embedded preview card (attachment), not a raw URL.
LinkedIn allows **one** link card per post — it comes from the first URL you paste,
and it only appears if no image is attached (the card itself will show your Medium
cover image, since Medium serves it as the article's preview image).

## How to post (2 minutes)

1. Paste the post text below (without any URL).
2. To make the tag real: delete `@Banan Alburaidi`, type `@`, start typing her
   name, and pick her profile
   ([linkedin.com/in/banan-alburaidi-557379242](https://www.linkedin.com/in/banan-alburaidi-557379242/))
   from the dropdown — a pasted name does not create a tag.
3. On a new line, paste the article URL:
   `https://medium.com/@ali.saadat81/supervisor-vs-blackboard-d866885bbcba`
   and wait a moment — the preview card appears below the text.
4. **Delete the URL line** — the card stays attached. Do NOT add an image
   (an image would replace the card).
5. Publish, then immediately add the first comment below (the repo link).
