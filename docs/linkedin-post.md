# LinkedIn post — Supervisor vs. Blackboard article

Both links go in the post body as full URLs. LinkedIn automatically shortens any
URL longer than 26 characters into an `lnkd.in/…` link when you publish — that is
exactly the style in the reference screenshot; there is nothing to do manually,
and you cannot pre-generate an `lnkd.in` link yourself.

## How to post

1. Paste the post text below with both full URLs.
2. Make the tag real: delete `@Banan Alburaidi`, type `@`, start typing her name,
   and pick her profile
   ([linkedin.com/in/banan-alburaidi-557379242](https://www.linkedin.com/in/banan-alburaidi-557379242/))
   from the dropdown — a pasted name does not create a tag.
3. LinkedIn will show a preview card for the first link. If you want text-only
   links (as in the screenshot), click the **X** on the card — the links stay in
   the text and get shortened on publish. Attaching your cover render image also
   removes the card and keeps both text links clickable.
4. Publish. Both URLs now display as `lnkd.in/…` short links.
